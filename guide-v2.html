Prosys OPC UA Simulation Server OT Zone v2

OPC UA simulator ที่รันบน mintserver (เครื่อง Linux Mint แยกต่างหาก) เปิดข้อมูล 307 tags ผ่าน OPC-UA protocol บน port 53530 ทุก 2 วินาที

เริ่มต้นอัตโนมัติด้วย systemd service เพราะ Prosys ใช้ install4j launcher ที่ fork Java แล้ว exit ทำให้ Type=simple คิดว่า service จบแล้ว — wrapper script ใช้ kill -0 $JAVA_PID loop เฝ้ารอแทน

# บน mintserver
sudo systemctl status prosys-opc
sudo systemctl start prosys-opc
sudo systemctl stop prosys-opc

InfluxDB — Flux Query ตัวอย่าง

// ดู fields ทั้งหมดที่มี
import "influxdata/influxdb/schema"
schema.fieldKeys(bucket: "opc-data",
  predicate: (r) => r._measurement == "opc_data")

// Boiler temperatures ทั้งหมด (20 ตัว)
from(bucket: "opc-data")
  |> range(start: -10m)
  |> filter(fn: (r) => r._measurement == "opc_data")
  |> filter(fn: (r) => strings.hasPrefix(v: r._field, prefix: "Temp_Boiler_"))
  |> aggregateWindow(every: 30s, fn: mean)

Trino — SQL Query

-- นับ records ต่อวัน
SELECT year, month, day, COUNT(*) AS records
FROM minio.opc.sensor_data
GROUP BY year, month, day
ORDER BY year, month, day DESC;

-- ดู schema (flat JSON fields)
DESCRIBE minio.opc.sensor_data;

-- Query sensor values
SELECT timestamp,
  json_extract_scalar(line, '$.Temp_Boiler_01') AS temp_boiler_01,
  json_extract_scalar(line, '$.Press_Line_01')  AS press_01
FROM minio.opc.sensor_data
WHERE year='2026' AND month='05'
ORDER BY timestamp DESC
LIMIT 20;
